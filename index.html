About Me

I'm a junior game developer focusing on game programming who has worked with both the Unity game engine and Unreal game engine, utilizing both C# and C++. Working on how game mechanics can be implemented using code and how they can be improved upon. What originally inspired me to become a game developer was my interest in taking the next step, so that instead of playing games, I could create them.
